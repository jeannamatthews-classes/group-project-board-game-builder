Welcome to BlueTile

This program allows you to create your own board game using a provided set of powerful, flexible editors.

Home menu. Create a game or play one you've already made!

About

BlueTile is a program that is designed to allow users to create their own simple board games, which can then be saved as files and loaded into the program to play or edit.

Unlike programs like Tabletop Simulator, which are designed to emulate the feeling of playing a physical board game and leave the enforcement of the rules up to the players, BlueTile has game creators write the rules for their games using an if-then-else sort of scripting language, and then during gameplay the rules of the game will be enforced by the program.

As of now, BlueTile does not currently have support for LAN hosting or servers, so when playing games with other people, you'll have to pass around a single device (or use screen sharing). BlueTile is only fully compatible with desktop/laptop/etc., not on mobile - playing games might work on mobile since the only method of control is clicks, but the game creation tools definitely don't all work on mobile.

Create Menu

To start building, click the Create button on the home menu screen. This will bring you to the Create Menu, where you can define the basics of your game state.

Determine your board shape, width, and height, as well as how many players your game will have.

The create menu where you shape your board and set up your game state.

The Editors

Click the edit button to toggle the Global, Board, Piece, Type, Button, and Title/Description editors. This is where you group logic and visuals into named components you can reuse across the board.

Add pieces, buttons, and types, as well as global variables and scripts. Click on components in the editors such as tiles and pieces to open their individual editors.

Use the editors to define and apply types.

Scripting Rules Editor

Once you've laid the groundwork, you can write scripting rules for your game elements using a simple if-then-else pseudo language. Each rule corresponds to an action or condition tied to objects like tiles, buttons, or pieces.

Add these rules to types, buttons, or global scripts. Then add types to pieces and tiles to define game logic, player interactions, and outcomes, giving you complete control over your game's behavior.

Editor for creating and customizing scripting rules.

The Layout Menu

After defining your types, use the preview icon to arrange the layout of active components like the board, buttons, title and description, and global variables.

Toggle, resize, and relocate layout windows. Or open window settings to set border color, border width, and background color to decorate your setup.

Final placement of game pieces and elements.

Gameplay

Once everything is ready, and you've saved your game file, head back to the main menu and click Play. You'll be prompted to upload the saved .json save code.

The game will play according to the rules you defined. Once a win condition is met, the Game Over screen will appear with the winner and an option to play again.

Playing a finished custom board game.
